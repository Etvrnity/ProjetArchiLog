## Dépendances du projet

Pour que notre application soit fonctionnelle, il faut intégrer plusieurs dépendances au projet :

- org.eclipse.angus:angus-mail:2.0.3 (service d'envoi d'e-mails)
- io.github.cdimascio:dotenv-java:3.0.0 (lecture sécurisé du fichier .env)
- com.mysql:mysql-connector-j:8.3.0 (driver de base de données MySQL)

_sur IntelliJ IDEA : File -> Project Structure -> Libraries -> New Project Library -> From Maven_
